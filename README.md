# Python Proxy Tools

- `ProxyDownloader` - downloads a list of proxies
- `ProxyChecker` - verifies the list of proxies


## File Info

- `sources.txt` - this contains the sources of the urls to download proxies from
- `https.txt` - this contains a list of https proxies to load
- `socks5` - this contains a list of socks5 proxies to load.
